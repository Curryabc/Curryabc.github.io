什么是WebService?

  • 百度百科：

    Web Service是一个平台独立的，低耦合的，自包含的、基于可编程的web的应用程序，可使用开放的XML（标准通用标记语言下的一个子集）标准来描述、发布、发现、协调和配置这些应用程序，用于开发分布式的互操作的应用程序。

  • WebService是两个系统的远程调用，使两个系统进行数据交互，如应用：天气预报服务、银行ATM取款、使用邮箱账号登录各网站等。
  • WebService之间的调用是跨语言的调用。Java、.Net、php，发送Http请求，使用的数据格式是XML格式。
  • “网络服务”（Web Service）的本质，就是通过网络调用其他网站的资源。
  • 这篇文章比较通俗的解释了WebService的含义，并且例举了一些生动的例子：http://www.ruanyifeng.com/blog/2009/08/what_is_web_service.html
